# surfriderpools

Marketing website for **Surfrider Pool Service**, a Dunedin, Florida pool company
serving Pinellas County and the Tampa Bay area.

Built per the `CalTech-Web/WEBSITETEMPLATE` conventions.

## Stack
- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- next/image
- Contact form via forms.caltechweb.com + Cloudflare Turnstile

## Structure
- `site/` - the Next.js application (Vercel Root Directory)
- `SITE_FACTS.md` - canonical content and business facts
- `CLAUDE.md` - project rules

## Develop
```
cd site
npm install
npm run dev
```

## Deploy
GitHub repo `CalTech-Web/surfriderpools`. Vercel project `surfriderpools`
under `brandon-7928s-projects`, Root Directory `site`,
production URL surfriderpools.vercel.app.
